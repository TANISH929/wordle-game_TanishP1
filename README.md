# Wordle Game in Python 

A simple Wordle-like game in Python.

## How to Play
- Run `python wordle.py`
- Guess the 5-letter word in 6 attempts
- Feedback:
  - `G` = correct letter, correct place
  - `Y` = correct letter, wrong place
  - `X` = letter not in the word

## Files
- `wordle.py` → game code
- `words.txt` → word list
